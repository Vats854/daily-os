# Project Card And Linked Tasks

## Что изменилось

- Название проекта редактируется прямо в заголовке.
- Критерий перехода и причина текущей стадии стали редактируемыми рабочими полями.
- В проекте можно создать новую связанную задачу.
- Существующую свободную задачу можно привязать к проекту.
- Связанную задачу можно завершить, вернуть, открыть в task core или отвязать.
- Все изменения сохраняются через общий `saveState()` и записываются в журнал действий.
- Внутренний индекс проектов сужен, чтобы рабочая карточка сохраняла полезную ширину.
